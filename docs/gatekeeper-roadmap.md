# Gatekeeper roadmap — approved 2026-07-02

Vision: engram as the universal, local-first gatekeeper of Derek's knowledge
and synthesis. Priorities below are post-wave-5/6 (wiki, nightly synthesis,
desktop app), in leverage order.

1. **Ask surface** — `engram ask` + UI mode + MCP tool: retrieve across
   wiki/dream/raw, synthesize ONE cited answer, every claim linking back to raw
   provenance. Answers, not results.
2. **Deliberate web capture** — a save gesture (browser extension or
   `engram save <url>`) snapshotting pages/highlights into raw_events. No
   firehose: "I decided this matters" produces a wiki; hoovering produces a
   dump. Bulk-inject + owner namespacing is the receiving end (built).
3. **Context injection** — the original coding-native wedge: SessionStart /
   post-checkout hook injecting "what you decided on this branch/project last
   time" into new Claude Code sessions; later, memory auto-attached to any
   agent via MCP. Push, not just pull.
4. **MCP distribution** — register the (built) server in Claude Desktop,
   claude.ai, Cursor. One memory layer under every AI tool.
5. **Trust machinery** — contradiction lint (wave 5), confidence markers,
   supersede-don't-delete corrections, and an answer-quality eval to pair with
   the retrieval benchmark.
6. **Demand-driven synthesis** — log queries locally; poorly-answered queries
   become the synthesis priority queue. Closes the self-managing loop.
7. **Survivability** — encrypted pg backups, wiki git-pushed to a private
   remote, documented restore path. Permanence is the point.

## Sequencing rule

Items 1–3 are wave 6/7 material; the rest slot behind. Do not parallelize the
vision — the pyramid proves itself on coding memory before it earns email,
Slack, or the browser.

## Positioning

Platform memory (Anthropic Kairos, ChatGPT memory) will compete for
"gatekeeper." Engram's defensible ground: **local-first, cross-tool,
user-owned, verbatim provenance** — platforms won't let memory leave their
walls. Win by being the one place that's yours, not by being everywhere first.
